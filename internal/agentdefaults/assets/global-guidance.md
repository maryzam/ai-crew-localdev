# Managed defaults

- Be brief and state the result first.
- Search before reading. Use rg or git grep and open only relevant files.
- Keep credentials out of prompts, logs, memory, and commits.
- Preserve unrelated work and ask before remote or destructive actions.
- Run focused checks first. Use ai-agent check for noisy commands.
- Keep memory local. Important rules belong in code, tests, hooks, or policy.
